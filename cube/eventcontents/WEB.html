Introduction

Have you ever got WEBBed?? If not, then here is the oppurtunity for you to get into a real trap of WEB.. So be prepared to come out of that trap.. start brushing your mind and prepare for a real challenge. An Apptitude cum web designing event is waiting to blow you mind..

Abstract



The Event consist of three levels as follows:-


Level 1:

*A precoded web page will be given to you which will be totally unreliable consisting bugs or errors.
*Each bug will be connected to a particular logical aptitude question.
*On cracking each question you will be provided with info which will help in resolving the bug.
*The teams debugging maximum errors will proceed to the next level.

Level 2:

A set of 5 logical question will be given to each team and the average time to solve those question would be around 5-6 minutes.
* By cracking question, the team will be given a password protected folder for each question consisting of several stuffs of same type(like images,or sound,or article ,or video) and you can use the web page of first level for finding codes for embedding the stuffs...
*The answer to the question will be the password for that folder..

Level 3:

*A theme will be given.
*then a time limit of 15 min will be given so that they can download or see the important things related to that theme
*Now sufficient time(approx 45 mins) will be given to make the web page on the theme as attractive as possible.
*The Top three webpages will be selected according to the attractiveness and the description about the given theme.
