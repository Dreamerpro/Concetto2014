Erreur Fixateur

Introduction

Were you one of those kids who relished the crunchy sound of a bug getting smashed? Well, then this is just the place you wanna be! Detect error in code, debug it and make the code feel alive. Exercise your brain cells and get ready for some errorful questions. If you think you are one of those computer geeks then this is the platform to prove yourself. Handling errorful code within stipulated time is not really an easy job. So, roll up your sleeves and get ready for this exciting competition

Abstract


*A team will consist of at most 2 members
*Your skills will be tested in 2 levels
*Each level will span for an hour
*No usage of any online resource or compilers,etc
*The codes would be presented in C programming language
*Registration will be done at http://www.concettoism.co.in before 17 Oct '14
*Any malpractice would lead to straight disqualification

Problem Description

First level will be Eliminating round and second will be Deciding round.
*Elimination level
This level will check your technical as well as aptitude skills. It will mainly consist of MCQs based on C programming language
*Deciding level
This level will decide your fate .The challenge is to make the code *bug* free without using any source. Your job would be to search the error, fix it and get the required output. It may consist of MCQs involving core knowledge in C programming language
Date-17 and 18 Oct'14.
